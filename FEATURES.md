# CSV HTML Editor - Features Guide

Welcome! This guide will help you understand how to use the CSV HTML Editor to update HTML content in your CSV files.

## What is this tool?

The CSV HTML Editor is a web-based tool that helps you edit HTML content inside CSV files. Instead of manually editing CSV files in a spreadsheet, you can visually edit HTML and see your changes in real-time.

## Getting Started

### Step 1: Upload Your CSV File

- **Drag and drop** your CSV file onto the upload area, or click to browse
- The tool will automatically detect your file and load it

### Step 2: Select Columns with HTML Content

- After uploading, you'll see a list of all columns in your CSV
- **Select the columns** that contain HTML content (you can select multiple)
- Click "Continue" to proceed

### Step 3: Start Editing

Once your file is loaded, you'll see three panels:

- **Left Panel**: List of all records (rows) in your CSV
- **Center Panel**: Editor where you can edit the HTML content
- **Right Panel**: List of all links found in the current record

## Tips for Content Editors

1. **Start with Search**: Use the search feature to quickly find the records you need to update

2. **Use Keyboard Navigation**: Arrow keys make it fast to move through records - much faster than clicking!

3. **Check the Links Panel**: Always check the right panel to see all links in a record before editing

4. **Save Frequently**: While changes auto-save, it's good practice to download your work periodically

5. **Load a Different File**: Use "Load different file" in the header if you need to switch files
